---
title: Point
description: API reference for points in Slate.
---

A `Point` represents a specific location in a Slate document. It consists of a path to a text node and an offset within that text node's content.

## Point API

### `get`

```ts
/**
 * Get the point from a location. If the location is a range, get the anchor
 * point (if `focus` is true, get the focus point). If the location is a path,
 * get the point at this path with offset 0.
 */
```

<APIParameters>
  <APIItem name="at" type="At | null" optional>
    The location to get the point from. Can be a Range, Point, or Path.
  </APIItem>
  <APIItem name="options" type="object" optional>
    <APISubList>
      <APISubListItem parent="options" name="focus" type="boolean" optional>
        If true and the location is a range, returns the focus point instead of the anchor point.
      </APISubListItem>
    </APISubList>
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `Point | undefined` - The point at the specified location, or `undefined` if not found.
</APIReturns>

### `transform`

Transform a point by an operation.

```ts
// Returns the transformed point, or null if the point was deleted
```

<APIParameters>
  <APIItem name="point" type="Point">
    The point to transform.
  </APIItem>
  <APIItem name="op" type="Operation">
    The operation to apply to the point.
  </APIItem>
  <APIItem name="options" type="PointTransformOptions" optional>
    <APISubList>
      <APISubListItem parent="options" name="affinity" type="TextDirection | null" optional>
        The direction to prefer when transforming the point.
      </APISubListItem>
    </APISubList>
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `Point | null` - The transformed point, or `null` if the point was deleted.
</APIReturns>

### `compare`

```ts
/**
 * Compare a point to another, returning an integer indicating whether the
 * point was before, at, or after the other.
 */
```

<APIParameters>
  <APIItem name="point" type="Point">
    The first point to compare.
  </APIItem>
  <APIItem name="another" type="Point">
    The second point to compare.
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `-1 | 0 | 1` - `-1` if before, `0` if at same location, `1` if after.
</APIReturns>

### `equals`

Check if a point is exactly equal to another.

<APIParameters>
  <APIItem name="point" type="Point">
    The first point to compare.
  </APIItem>
  <APIItem name="another" type="Point">
    The second point to compare.
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `boolean` - `true` if the points are equal, `false` otherwise.
</APIReturns>

### `isAfter`

Check if a point is after another.

<APIParameters>
  <APIItem name="point" type="Point">
    The point to check.
  </APIItem>
  <APIItem name="another" type="Point">
    The point to compare against.
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `boolean` - `true` if the first point is after the second, `false` otherwise.
</APIReturns>

### `isBefore`

Check if a point is before another.

<APIParameters>
  <APIItem name="point" type="Point">
    The point to check.
  </APIItem>
  <APIItem name="another" type="Point">
    The point to compare against.
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `boolean` - `true` if the first point is before the second, `false` otherwise.
</APIReturns>

### `isPoint`

Check if a value implements the `Point` interface.

<APIParameters>
  <APIItem name="value" type="any">
    The value to check.
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `boolean` - `true` if the value is a point, `false` otherwise.
</APIReturns>

## Types

### `Point`

```ts
/**
 * A Point represents a specific location in a Slate document. It consists of a
 * path to a text node and an offset within that text node's content.
 */
type Point = {
  /** The index of the character in the text node. */
  offset: number;
  /** The path to the text node. */
  path: Path;
};
```

<APIAttributes>
  <APIItem name="offset" type="number">
    The index of the character in the text node.
  </APIItem>
  <APIItem name="path" type="Path">
    The path to the text node.
  </APIItem>
</APIAttributes>

### `PointTransformOptions`

```ts
interface PointTransformOptions {
  affinity?: TextDirection | null;
}
```

<APIAttributes>
  <APIItem name="affinity" type="TextDirection | null" optional>
    The direction to prefer when transforming the point.
  </APIItem>
</APIAttributes>

### `PointEntry`

```ts
/**
 * `PointEntry` objects are returned when iterating over `Point` objects that
 * belong to a range.
 */
type PointEntry = [Point, 'anchor' | 'focus'];
```

<APIAttributes>
  <APIItem name="[0]" type="Point">
    The point location.
  </APIItem>
  <APIItem name="[1]" type="'anchor' | 'focus'">
    Indicates whether this point is the anchor or focus of a range.
  </APIItem>
</APIAttributes>
