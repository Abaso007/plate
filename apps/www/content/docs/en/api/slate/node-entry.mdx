## Types

### `NodeEntry`

Node entry objects are returned when iterating over the nodes in a Slate document tree. They consist of the node and its `Path` relative to the root node in the document.

<APIParameters>
  <APIItem name="NodeEntry" type="[Node, Path]">
    <APISubList>
      <APISubListItem>
        First element: The node itself
      </APISubListItem>
      <APISubListItem>
        Second element: The path to the node
      </APISubListItem>
    </APISubList>
  </APIItem>
</APIParameters>
