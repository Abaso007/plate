---
title: Path
description: API reference for paths in Slate.
---

`Path` arrays are a list of indexes that describe a node's exact position in a Slate node tree. Although they are usually relative to the root `Editor` object, they can be relative to any `Node` object.

## Path API

### `operationCanTransformPath`

Check if an operation can transform a path.

<APIParameters name="operationCanTransformPath">
  <APIItem name="operation" type="Operation<N>">
    The operation to check.
  </APIItem>
</APIParameters>

<APIReturns name="operationCanTransformPath">
  Returns `boolean` - `true` if the operation is an insert, merge, move, remove, or split operation.
</APIReturns>

### `transform`

Transform a path by an operation.

<APIParameters name="transform">
  <APIItem name="path" type="Path">
    The path to transform.
  </APIItem>
  <APIItem name="operation" type="Operation">
    The operation to apply.
  </APIItem>
  <APIItem name="options" type="PathTransformOptions" optional>
    <APISubList>
      <APISubListItem parent="options" name="affinity" type="TextDirection | null" optional>
        The affinity of the transform.
      </APISubListItem>
    </APISubList>
  </APIItem>
</APIParameters>

<APIReturns name="transform">
  Returns `Path | null` - The transformed path, or `null` if the path was deleted.
</APIReturns>

### `ancestors`

Get a list of ancestor paths for a given path.

<APIParameters name="ancestors">
  <APIItem name="path" type="Path">
    The path to get ancestors for.
  </APIItem>
  <APIItem name="options" type="PathAncestorsOptions" optional>
    <APISubList>
      <APISubListItem parent="options" name="reverse" type="boolean" optional>
        If true, the paths are returned in reverse order (deepest to shallowest).
      </APISubListItem>
    </APISubList>
  </APIItem>
</APIParameters>

<APIReturns name="ancestors">
  Returns `Path[]` - An array of paths sorted from shallowest to deepest ancestor (unless reversed).
</APIReturns>

### `child`

Get a path to a child at the given index.

<APIParameters name="child">
  <APIItem name="path" type="Path">
    The parent path.
  </APIItem>
  <APIItem name="index" type="number">
    The child index.
  </APIItem>
</APIParameters>

<APIReturns name="child">
  Returns `Path` - The path to the child node.
</APIReturns>

### `common`

Get the common ancestor path of two paths.

<APIParameters name="common">
  <APIItem name="path" type="Path">
    The first path.
  </APIItem>
  <APIItem name="another" type="Path">
    The second path.
  </APIItem>
</APIParameters>

<APIReturns name="common">
  Returns `Path` - The common ancestor path.
</APIReturns>

### `compare`

Compare a path to another, returning an integer indicating whether the path was before, at, or after the other.

<APIParameters name="compare">
  <APIItem name="path" type="Path">
    The first path to compare.
  </APIItem>
  <APIItem name="another" type="Path">
    The second path to compare.
  </APIItem>
</APIParameters>

<APIReturns name="compare">
  Returns `-1 | 0 | 1` - `-1` if before, `0` if at same location, `1` if after.
</APIReturns>

### `endsAfter`

Check if a path ends after one of the indexes in another.

<APIParameters name="endsAfter">

<APIReturns name="endsAfter">

## Types

### `Path`

An array of numbers representing the indexes to traverse to reach a specific node in the document tree.

### `PathAncestorsOptions`

Options for getting ancestor paths.

<APIAttributes name="PathAncestorsOptions">
  <APIItem name="reverse" type="boolean" optional>
    If true, the paths are returned in reverse order.
  </APIItem>
</APIAttributes>

### `PathLevelsOptions`

Options for getting path levels.

<APIAttributes name="PathLevelsOptions">
  <APIItem name="reverse" type="boolean" optional>
    If true, the paths are returned in reverse order.
  </APIItem>
</APIAttributes>

### `PathTransformOptions`

Options for transforming paths.

<APIAttributes name="PathTransformOptions">
  <APIItem name="affinity" type="TextDirection | null" optional>
    The affinity of the transform.
  </APIItem>
</APIAttributes>
