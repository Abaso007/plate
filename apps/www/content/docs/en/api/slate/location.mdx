### `isAt`

Check if a value implements the `At` interface.

<APIParameters>
  <APIItem name="value" type="any">
    The value to check.
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `boolean` - `true` if the value is either a Location or a Node.
</APIReturns>

### `isLocation`

Check if a value implements the `Location` interface.

<APIParameters>
  <APIItem name="value" type="any">
    The value to check.
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `boolean` - `true` if the value is a Path, Point, or Range.
</APIReturns>

### `isSpan`

Check if a value implements the `Span` interface.

<APIParameters>
  <APIItem name="value" type="any">
    The value to check.
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `boolean` - `true` if the value is a valid Span.
</APIReturns>

## Types

### `TLocation`

The `TLocation` interface is a union of the ways to refer to a specific location in a Slate document: paths, points or ranges.

<APIParameters>
  <APIItem name="TLocation" type="Path | Point | TRange">
    <APISubList>
      <APISubListItem>
        Path: An array of numbers representing a node's position
      </APISubListItem>
      <APISubListItem>
        Point: An object with `path` and `offset` properties
      </APISubListItem>
      <APISubListItem>
        TRange: An object with `anchor` and `focus` points
      </APISubListItem>
    </APISubList>
  </APIItem>
</APIParameters>

### `Location`

The `Location` interface is a union of the ways to refer to a specific location in a Slate document: paths, points or ranges.

<APIParameters>
  <APIItem name="Location" type="Path | Point | Range">
    <APISubList>
      <APISubListItem>
        Path: An array of numbers representing a node's position
      </APISubListItem>
      <APISubListItem>
        Point: An object with `path` and `offset` properties
      </APISubListItem>
      <APISubListItem>
        Range: An object with `anchor` and `focus` points
      </APISubListItem>
    </APISubList>
  </APIItem>
</APIParameters>

### `Span`

The `Span` interface is a low-level way to refer to locations in nodes without using `Point` which requires leaf text nodes to be present.

<APIParameters>
  <APIItem name="Span" type="[Path, Path]">
    A tuple containing two paths.
  </APIItem>
</APIParameters>
