### `ancestor`

Get the node at a specific path, asserting that it's an ancestor node.

<APIParameters>
  <APIItem name="root" type="TNode">
    The root node to start from.
  </APIItem>
  <APIItem name="path" type="Path">
    The path to the ancestor node.
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `N | undefined` - The ancestor node if found, `undefined` otherwise.
</APIReturns>

### `ancestors`

Return a generator of all the ancestor nodes above a specific path.

<APIParameters>
  <APIItem name="root" type="TNode">
    The root node to start from.
  </APIItem>
  <APIItem name="path" type="Path">
    The path to get ancestors for.
  </APIItem>
  <APIItem name="options" type="NodeAncestorsOptions" optional>
    <APISubList>
      <APISubListItem parent="options" name="reverse" type="boolean" optional>
        If true, returns ancestors top-down instead of bottom-up.
      </APISubListItem>
    </APISubList>
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `Generator<NodeEntry<N>, void, undefined>` - A generator of node entries.
</APIReturns>

## Types

### `TNode`

The `TNode` union type represents all of the different types of nodes that occur in a Slate document tree.

```ts
type TNode = Editor | TElement | TText;
```

### `Descendant`

The `Descendant` union type represents nodes that are descendants in the tree.

```ts
type Descendant = TElement | TText;
```

### `Ancestor`

The `Ancestor` union type represents nodes that are ancestors in the tree.

```ts
type Ancestor = Editor | TElement;
```

### `Node`

The `Node` union type represents all of the different types of nodes that occur in a Slate document tree.

```ts
type Node = TNode;
```
