---
title: Editor Transforms
description: API reference for editor transformation operations in Slate.
---

Editor transforms are methods that modify the editor state. They are grouped into several categories: node operations, text operations, selection operations, DOM operations, and history operations.

## `editor.tf`

### `insertFragment`

Insert a fragment of nodes at the specified location or current selection.

<APIParameters>
  <APIItem name="fragment" type="N[]">
    The fragment of nodes to insert.
  </APIItem>
  <APIItem name="options" type="InsertFragmentOptions" optional>
    <APISubList>
      <APISubListItem parent="options" name="at" type="At" optional>
        The location to insert at. Defaults to current selection.
      </APISubListItem>
      <APISubListItem parent="options" name="batchDirty" type="boolean" optional>
        Whether to batch dirty operations.
      </APISubListItem>
      <APISubListItem parent="options" name="hanging" type="boolean" optional>
        Whether the insertion is hanging.
      </APISubListItem>
      <APISubListItem parent="options" name="voids" type="boolean" optional>
        Whether to allow insertion in void nodes.
      </APISubListItem>
    </APISubList>
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `void`
</APIReturns>

### `wrapNodes`

Wrap nodes at the specified location in an element container.

<APIParameters>
  <APIItem name="element" type="N">
    The element to wrap the nodes with.
  </APIItem>
  <APIItem name="options" type="WrapNodesOptions<V>" optional>
    <APISubList>
      <APISubListItem parent="options" name="at" type="At" optional>
        The location to wrap nodes at. Defaults to current selection.
      </APISubListItem>
      <APISubListItem parent="options" name="children" type="boolean" optional>
        When true, wrap node children into a single element:
        - Wraps the first child node into the element
        - Move the other child nodes next to the element children
      </APISubListItem>
      <APISubListItem parent="options" name="hanging" type="boolean" optional>
        Whether the operation is hanging.
      </APISubListItem>
      <APISubListItem parent="options" name="match" type="NodeMatch<V>" optional>
        Predicate function to filter which nodes to wrap.
      </APISubListItem>
      <APISubListItem parent="options" name="mode" type="'highest' | 'lowest'" optional>
        The matching mode to use when finding nodes.
      </APISubListItem>
      <APISubListItem parent="options" name="split" type="boolean" optional>
        Whether to allow splitting nodes during wrapping. For example, if `ipsum` was selected in a `Text` node with `lorem ipsum dolar`, `split: true` would wrap the word `ipsum` only, resulting in splitting the `Text` node. If `split: false`, the entire `Text` node `lorem ipsum dolar` would be wrapped.
      </APISubListItem>
      <APISubListItem parent="options" name="voids" type="boolean" optional>
        Whether to allow wrapping void nodes.
      </APISubListItem>
    </APISubList>
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `void`
</APIReturns>
