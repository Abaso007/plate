---
title: Text
description: API reference for text nodes in Slate.
---

`Text` objects represent the nodes that contain the actual text content of a Slate document along with any formatting properties. They are always leaf nodes in the document tree as they cannot contain any children.

## Text API

### `decorations`

Get the leaves for a text node given decorations.

+ ```ts
+ // Returns an array of text nodes with the decorations applied
+ ```

<APIParameters>
  <APIItem name="node" type="TText">
    The text node to get leaves from.
  </APIItem>
  <APIItem name="decorations" type="DecoratedRange[]">
    Array of decorated ranges to apply.
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `TText[]` - An array of text nodes with the decorations applied.
</APIReturns>

### `equals`

Check if two text nodes are equal.

+ ```ts
+ // When loose=true, only compares formatting properties, not text content
+ // This is useful for checking if sibling text nodes can be merged
+ ```

<APIParameters>
  <APIItem name="text" type="TText">
    First text node to compare.
  </APIItem>
  <APIItem name="another" type="TText">
    Second text node to compare.
  </APIItem>
  <APIItem name="options" type="TextEqualsOptions" optional>
    <APISubList>
      <APISubListItem parent="options" name="loose" type="boolean" optional>
        If true, the text content is not compared. This is used to check whether sibling text nodes can be merged.
      </APISubListItem>
    </APISubList>
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `boolean` - `true` if the text nodes are equal according to the comparison rules, `false` otherwise.
</APIReturns>

### `isText`

Check if a value implements the `Text` interface.

<APIParameters>
  <APIItem name="value" type="any">
    The value to check.
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `boolean` - `true` if the value is a valid text node, `false` otherwise.
</APIReturns>

### `isTextList`

Check if a value is a list of `Text` objects.

<APIParameters>
  <APIItem name="value" type="any">
    The value to check.
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `boolean` - `true` if the value is an array of text nodes, `false` otherwise.
</APIReturns>

### `isTextProps`

Check if some props are a partial of Text.

<APIParameters>
  <APIItem name="props" type="any">
    The props to check.
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `boolean` - `true` if the props match a partial text node structure, `false` otherwise.
</APIReturns>

### `matches`

Check if a text matches set of properties.

```ts
// Note: this is for matching custom properties only
// Does not ensure that the `text` property of two nodes are equal
```

<APIParameters>
  <APIItem name="text" type="TText">
    The text node to check.
  </APIItem>
  <APIItem name="props" type="Partial<TText>">
    The properties to match against.
  </APIItem>
</APIParameters>

<APIReturns>
  Returns `boolean` - `true` if the text node matches the properties, `false` otherwise.
</APIReturns>

## Types

### `Text`

- The base text node type that contains the actual text content and any formatting properties.
+ ```ts
+ /**
+  * `Text` objects represent the nodes that contain the actual text content of a
+  * Slate document along with any formatting properties. They are always leaf
+  * nodes in the document tree as they cannot contain any children.
+  */
+ ```

<APIAttributes>
  <APIItem name="text" type="string">
    The text content of the node.
  </APIItem>
  <APIItem name="[key: string]" type="any">
    Additional formatting properties.
  </APIItem>
</APIAttributes>

### DecoratedRange

Represents a range with decorations applied.

+ ```ts
+ // A range that has decorations applied to it
+ // Imported from slate: type DecoratedRange = SlateDecoratedRange
+ ```

### TextEqualsOptions

Options for comparing text nodes.

+ ```ts
+ interface TextEqualsOptions {
+   /**
+    * If true, the text is not compared. This is used to check whether sibling
+    * text nodes can be merged.
+    */
+   loose?: boolean;
+ }
+ ```

<APIAttributes>
  <APIItem name="loose" type="boolean" optional>
    If true, the text content is not compared. Used to check if sibling text nodes can be merged.
  </APIItem>
</APIAttributes>

### TextOf<N>

A utility type to get all the text node types from a root node type.

+ ```ts
+ /**
+  * A utility type to get all the text node types from a root node type.
+  */
+ type TextOf<N extends TNode> = Editor extends N
+   ? TText
+   : TElement extends N
+     ? TText
+     : N extends Editor
+       ? TextOf<N['children'][number]>
+       : N extends TElement
+         ? Extract<N['children'][number], TText> | TextOf<N['children'][number]>
+         : N extends TText
+           ? N
+           : never;
+ ```

### TextIn<V>

+ A utility type to get all text node types from a Value type.
+ 
+ ```ts
+ type TextIn<V extends Value> = TextOf<V[number]>;
+ ```

### MarksOf<N>

A utility type to get all the mark types from a root node type.

+ ```ts
+ type MarksOf<N extends TNode> = Simplify<
+   UnionToIntersection<NodeProps<TextOf<N>>>
+ >;
+ ```

### MarksIn<V>

+ A utility type to get all mark types from a Value type.
+ 
+ ```ts
+ type MarksIn<V extends Value> = MarksOf<V[number]>;
+ ```

### MarkKeysOf<N>

+ A utility type to get all possible mark keys from a node type.
+ 
+ ```ts
+ type MarkKeysOf<N extends TNode> =
+   {} extends MarksOf<N> ? unknown : keyof MarksOf<N>;
+ ```
