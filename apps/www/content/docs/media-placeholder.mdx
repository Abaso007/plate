## Features

- Displays placeholder for media elements (image, video, audio, file).
- Allows typing URLs or uploading local files directly in the placeholder.
- Customizable placeholder text and styles for different media types.
- Automatically converts to the appropriate media element when URL is entered or file is uploaded.
- Improves user experience by providing a seamless way to insert media content.
- Supports drag and drop functionality for easy file uploads.
- Automatically validates URLs and file types for each media category.

</PackageInfo>

## Installation


```bash
npm install @udecode/plate-callout
```

## Usage

```tsx
// ...
import {
  AudioPlugin,
  FilePlugin,
  ImagePlugin,
  MediaEmbedPlugin,
  PlaceholderPlugin,
  VideoPlugin,
} from '@udecode/plate-media/react';

const editor = usePlateEditor({
  id: 'callout-demo',
  override: {
    components: {
      ...otherComponents,
      [PlaceholderPlugin.key]: MediaPlaceholderElement,
    },
  },
  plugins: [
    ...otherPlugins,
    PlaceholderPlugin,
  ],
  value: calloutValue,
});
```
