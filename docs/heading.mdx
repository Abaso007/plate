## Features

- Create headings of various levels (H1 to H6) to structure content
- Keyboard shortcuts for quick heading level changes
- Configurable heading levels support
- Renders as appropriate HTML heading tags (`<h1>` to `<h6>`) by default

</PackageInfo>

## Installation

```bash
npm install @udecode/plate-basic-elements
```

## Usage

### Individual Plugins (Recommended)

```tsx
import { H1Plugin, H2Plugin, H3Plugin } from '@udecode/plate-basic-elements/react';

const plugins = [
  // ...otherPlugins,
  H1Plugin.configure({
    shortcuts: { toggle: { keys: 'mod+alt+1' } },
  }),
  H2Plugin.configure({
    shortcuts: { toggle: { keys: 'mod+alt+2' } },
  }),
  H3Plugin.configure({
    shortcuts: { toggle: { keys: 'mod+alt+3' } },
  }),
];
```

### HeadingPlugin

Alternative to individual plugins, use `HeadingPlugin` to include all heading levels at once:

```tsx
import { HeadingPlugin } from '@udecode/plate-basic-elements/react';

const plugins = [
  // ...otherPlugins,
  HeadingPlugin.configure({
    options: { levels: 3 }, // Only H1, H2, H3
  }),
];
```

### With Custom Components

```tsx
import { H1Plugin, H2Plugin, H3Plugin } from '@udecode/plate-basic-elements/react';
import { H1Element, H2Element, H3Element } from '@/components/heading-element';

const plugins = [
  // ...otherPlugins,
  H1Plugin.configure({
    node: { component: H1Element },
    shortcuts: { toggle: { keys: 'mod+alt+1' } },
  }),
  H2Plugin.configure({
    node: { component: H2Element },
    shortcuts: { toggle: { keys: 'mod+alt+2' } },
  }),
  H3Plugin.configure({
    node: { component: H3Element },
    shortcuts: { toggle: { keys: 'mod+alt+3' } },
  }),
];
```

## Plugins

### H1Plugin

- `H1Plugin` - Creates H1 headings
- `H2Plugin` - Creates H2 headings  
- `H3Plugin` - Creates H3 headings
- `H4Plugin` - Creates H4 headings
- `H5Plugin` - Creates H5 headings
- `H6Plugin` - Creates H6 headings

### `HeadingPlugin`

Alternative to the individual plugins, `HeadingPlugin` includes all heading levels (H1 to H6) by default.

<API name="HeadingPlugin">
<APIOptions>
  <APIItem name="levels" type="number" optional>
    Specifies the heading levels supported, ranging from 1 to the specified levels.
    - **Default:** `6`
  </APIItem>
</APIOptions>
</API>

## Transforms

### `editor.tf.h1.toggle`

Toggles the current block between the specified heading level and a paragraph.

```tsx
// Toggle H1
editor.tf.h1.toggle();

// Toggle H2  
editor.tf.h2.toggle();

// Toggle H3
editor.tf.h3.toggle();

// And so on for H4, H5, H6...
```
