## Features

- Provides text alignment options: left, right, center, or justify.

</PackageInfo>

## Installation

```bash
npm install @udecode/plate-basic-styles
```

## Usage

```tsx
import { KEYS } from '@udecode/plate';
import { createPlateEditor } from '@udecode/plate/react';
import { TextAlignPlugin } from '@udecode/plate-basic-styles/react';

const editor = createPlateEditor({
  plugins: [
    TextAlignPlugin.configure({
      inject: {
        targetPlugins: [
          KEYS.p,
          ...KEYS.heading,
        ],
      },
    }),
  ],
});
```

## Plugins

### `TextAlignPlugin`

Plugin for aligning text within block elements.

## Transforms

### `tf.textAlign.setNodes`

Sets the alignment for the specified block elements in the editor.

<API name="tf.textAlign.setNodes">
<APIParameters>
<APIItem name="value" type="Alignment">
The alignment value.

</APIParameters>
</API>
