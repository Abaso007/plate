## 功能

- 字体大小、字体、颜色、背景颜色和粗细。

## 插件

- `FontFamilyPlugin`: 使用带有 `font-family` 样式的内联元素更改字体
- `FontSizePlugin`: 使用 CSS 类或 `font-size` 样式控制字体大小
- `FontColorPlugin`: 使用 `color` 样式控制字体颜色
- `FontBackgroundColorPlugin`: 使用 `background-color` 样式控制背景颜色
- `FontWeightPlugin`: 使用 `font-weight` 样式控制字体粗细

</PackageInfo>

## 安装

```bash
npm install @udecode/plate-basic-styles
```

## 使用

```tsx
import {
  FontBackgroundColorPlugin,
  FontColorPlugin,
  FontSizePlugin,
} from '@udecode/plate-basic-styles/react';

const plugins = [
  // ...其他插件
  FontColorPlugin,
  FontBackgroundColorPlugin,
  FontSizePlugin,
];
```

## 插件

### `FontBackgroundColorPlugin`

用于字体背景颜色格式化的插件。

### `FontColorPlugin`

用于字体颜色格式化的插件。

### `FontFamilyPlugin`

用于字体格式化的插件。

### `FontSizePlugin`

用于字体大小格式化的插件。

#### API

##### `api.fontSize.setMark`

设置字体大小标记。

<API name="setMark">

#### 工具

##### `toUnitLess`

将字体大小值转换为无单位值。

</API>
