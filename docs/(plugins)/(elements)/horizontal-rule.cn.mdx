## 功能

- 插入水平线以分隔内容或指示主题转换
- 在新行开头输入三个破折号 (`---`) 以转换为水平分割线

</PackageInfo>

## 安装

```bash
npm install @udecode/plate-basic-nodes
```

## 使用

```tsx
import { insertNodes, setNodes, KEYS } from '@udecode/plate';
import { AutoformatPlugin } from '@udecode/plate-autoformat/react';
import { HorizontalRulePlugin } from '@udecode/plate-basic-nodes/react';
import { SelectOnBackspacePlugin } from '@udecode/plate/react';

const plugins = [
  // ...其他插件,
  HorizontalRulePlugin,
  SelectOnBackspacePlugin.configure({
    options: { query: { allow: [KEYS.hr] } },
  }),
  AutoformatPlugin.configure({
    options: {
      rules: [
        {
          mode: 'block',
          type: KEYS.hr,
          match: ['---', '—-', '___ '],
          format: (editor) => {
            setNodes(editor, { type: KEYS.hr });
            insertNodes(editor, {
              type: KEYS.p,
              children: [{ text: '' }],
            });
          },
        },
      ],
    },
  }),
];
```

## 插件

### `HorizontalRulePlugin`

用于插入水平分割线以分隔内容的插件。
