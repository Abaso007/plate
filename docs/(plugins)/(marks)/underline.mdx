## Features

- Apply underline formatting to text
- Keyboard shortcut support for quick formatting
- Renders as `<u>` HTML element by default

</PackageInfo>

## Installation

```bash
npm install @udecode/plate-basic-nodes
```

## Usage

```tsx
import { UnderlinePlugin } from '@udecode/plate-basic-nodes/react';

const plugins = [
  // ...otherPlugins,
  UnderlinePlugin,
];
```

## Plugins

### `UnderlinePlugin`

Plugin for underline text formatting. Renders as `<u>` HTML element by default.

## Transforms

### `tf.underline.toggle`

Toggles the underline formatting for the selected text.

Default Shortcut: `Cmd + U`
