## 功能

- 提供文本对齐选项：左对齐、右对齐、居中对齐或两端对齐。

</PackageInfo>

## 安装

```bash
npm install @platejs/basic-styles
```

## 使用

```tsx
import { createPlateEditor, KEYS } from 'platejs/react';
import { TextAlignPlugin } from '@platejs/basic-styles/react';
import { HeadingPlugin } from '@platejs/basic-nodes/react';

const editor = createPlateEditor({
  plugins: [
    HeadingPlugin,
    TextAlignPlugin.configure({
      inject: {
        targetPlugins: [
          KEYS.p,
          KEYS.h1,
        ],
      },
    }),
  ],
});
```

## 插件

### `TextAlignPlugin`用于在块元素中对齐文本的插件。
