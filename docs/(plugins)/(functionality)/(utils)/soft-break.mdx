## Features

- Allows insertion of line breaks within a text block without starting a new block
- Configurable per plugin using the `breakMode` property
- Multiple modes: `'lineBreak'`, `'splitOnEmptyLine'`, or `'split'` (default)

</PackageInfo>

## Usage

Enable soft break behavior for specific plugin types by setting `breakMode` in their node configuration:

```tsx
import { BlockquotePlugin } from '@udecode/plate-blockquote/react';

const plugins = [
  // ...otherPlugins,
  BlockquotePlugin.configure({
    node: {
      breakMode: 'lineBreak',
    },
  }),
];
```

### Modes

- **`'lineBreak'`**: Always inserts a line break when Enter is pressed within this node type
- **`'splitOnEmptyLine'`**: Inserts a line break normally, but splits the block when on an empty line (when the previous character is `\n`)
- **`'split'`** or `undefined`: Standard behavior - splits the block (default)

```tsx
// Always insert line breaks in blockquotes
BlockquotePlugin.configure({
  node: { breakMode: 'lineBreak' },
});

// Insert line breaks in code blocks, but split when on empty lines
CodeBlockPlugin.configure({
  node: { breakMode: 'splitOnEmptyLine' },
});
```

## Customization

For more complex break behavior, you can override the `insertBreak` transform:

```tsx
const plugins = [
  // ...otherPlugins,
  MyPlugin.overrideEditor(
    ({
      editor,
      tf: { insertBreak },
    }) => ({
      transforms: {
        insertBreak: () => {
          // Custom break logic here
          if (/* your custom condition */) {
            editor.tf.insertSoftBreak();
            return;
          }

          // Default behavior
          insertBreak();
        },
      },
    })
  ),
];
```
